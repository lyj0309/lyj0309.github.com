模块配置

可用模块 filebeat modules list

开启模块 filebeat modules enable system nginx mysql

模块设置路径 ： /var/filebeat/modules.d/

启动

验证服务 filebeat setup -e

验证完可用后，后台启动 sudo service filebeat start
