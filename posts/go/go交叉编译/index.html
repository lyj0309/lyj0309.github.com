Go 交叉编译

go 语言再啥平台都支持交叉编译，值得注意的是 cgo，在一些 linux 发行版中用的从语言库 go 并不支持，所以不能使用 cgo，再一些情况下 cgo 是可以提升运行速度的

使用

选择需要编译的系统

编译 linux go env -w GOOS=linux

编译 windows go env -w GOOS=windows

###选择需要编译的 cpu 架构

go env -w GOARCH=amd64

go env -w GOARCH=arm64
